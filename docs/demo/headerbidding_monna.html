Header bidding monna example

Using Prebid.JS 9.25+

Add the RetailSpot bidder in your prebid configuration:

      
        ...
        bids: [{
            bidder: 'retailspot',
            params: {
              placement: 'eq-609785-1856964-125234',
              format: "slider",
              widget: "monna",
              height: 350,
              width: 200,
              // allows slider format to be shown outside of predib target div
              target: null,
              // Add more rsplayer configuration here...
            }
        }]
